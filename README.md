# 🖥️ Computer Vision — Object Detection & Video Processing

Advanced Computer Vision projects implementing object detection, 
video processing, and deep learning models using OpenCV and Python.

## 📁 Projects Included

| 📓 Notebook | 📝 Description |
|---|---|
| `COMPUTERvision.ipynb` | Core Computer Vision algorithms & techniques |
| `objectdetection.ipynb` | Real-time Object Detection using Deep Learning |
| `video.ipynb` | Video processing & frame analysis |

## ✨ Features
- 🎯 Real-time object detection
- 🎥 Video frame processing & analysis
- 🖼️ Image processing techniques
- 🧠 Deep learning based CV models
- 📊 CNN architecture implementation
